## Size Reduction

- Eliminated 3 redundant requirements files

**Result**: Much cleaner, easier-to-maintain codebase with single source of truth for dependencies

## How to Install Clean Dependencies

```bash
# Using pip
pip install -r requirements.txt

# Or using uv (faster)
uv pip install -r requirements.txt
```

## Running the Application

```bash
# Activate virtual environment
source .venv/bin/activate

# Run migrations (if needed)
python manage.py migrate

# Start Daphne server
daphne -b 127.0.0.1 -p 8000 AMS.asgi:application
```
